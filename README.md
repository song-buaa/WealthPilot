# WealthPilot

个人资产管理应用 - 智能财富规划助手

## 项目简介

WealthPilot 是一个智能化的个人资产管理应用，帮助用户进行资产配置、投资分析和财务规划。

## 版本历史

| 版本 | 说明 |
|------|------|
| v1.3 | 当前最新版本 - 代码质量优化，增加测试 |
| v1.2 | 重构版本 - 模块化架构，增加 pages |
| v1.1 | 功能修复版本 |
| v1.0 | 初始版本 |

## 技术栈

- Python
- Streamlit
- AI 模型集成（Kimi、Claude、DeepSeek）
- Pytest（测试框架）

## 快速开始

1. 进入最新版本目录：
   ```bash
   cd M1_v1.3
   ```

2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 运行应用：
   ```bash
   streamlit run streamlit_app.py
   ```

4. 运行测试：
   ```bash
   pytest
   ```

## 开发协作

- **GPT** - 功能设计
